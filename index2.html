General

El soporte técnico o asistencia técnica es un rango de servicios por medio del cual se proporciona asistencia a los usuarios al tener algún problema al utilizar un producto o servicio.

Navegacion

Cuando el usuario se comunica con problemas de navegacion, ya sea que no navega o que navega lento, seguir el siguiente paso a paso:

  • Verificar que tenga Datos
  • Verificar que tenga los datos encendidos
  • Configurar APN: movilexito.net.co
  • Configurar Modo de Red: GSM/WCDMA
  • Validar que el IMEI no se encuentre bloqueado
  • Probar en otro equipo
  • Escalar el caso

Por que no Navego en 4G?

  • El celular debe soportar 4G
  • La banda de 4g es (1700/2100)
  • Configurar APN: movilexito.net.co
  • Configurar Modo de Red: LTE/GSM/WCDMA

Llamadas

Cuando el usuario llama, ya sea porque no recibe o no puede realizar llamadas, realizar el siguiente paso a paso:

  • Verificar que la linea este activa
  • Validar si el problema es de llamadas de entrada o salida
  • Brindar codigos de restriccion o desvio
  • Validar que el IMEI no se encuentre bloqueado
  • Probar en otro equipo
  • Escalar el caso

Consumos Altos

Lo primero que debes tener presente es que, los precios de datos tarifan por consumo de Kilobyte (KB), ya sea enviado o recibido, por lo que los consumos varían según cada persona.

Para establecer una aproximación de los datos que se consumen tenemos los siguientes valores de las actividades que un usuario realiza en general: enviar por e-mail una foto, 600 KB; descargar la aplicación de un juego, 3 o 4 MB; enviar y recibir mensajes por whatsapp, 5 MB; remitir un e-mail de texto, 100 KB; descargar un archivo word sin imágenes, 250 KB; visitar un sitio web sin o con pocas imágenes, 500 KB: descargar una canción, 3 MB: reproducir un video, 10 MB; visualizar mapas mientras se navega con GPS,1 MB cada 20Km
